## 📊 游戏规则

### 基本规则

1. **移动规则**
   - 坦克可以在四个方向移动
   - 不能穿墙移动
   - 移动受网格限制

2. **战斗规则**
   - 发射炮弹消灭敌方坦克
   - 被敌方炮弹击中失去生命
   - 生命归零游戏结束

3. **得分规则**
   - 消灭敌方坦克：+10 分
   - 保护基地：额外奖励

## 💡 游戏技巧

### 新手提示
1. **利用掩体** - 利用墙壁躲避炮弹
2. **预判轨迹** - 提前判断敌人位置
3. **保护基地** - 不要让敌人接近基地
4. **节省弹药** - 不要盲目射击

### 高级技巧
1. **快速反应** - 快速躲避敌方炮弹
2. **精准射击** - 提高命中率
3. **游击战术** - 移动中攻击敌人
4. **团队配合** - 保护友方坦克

### 策略建议
- 🎯 **先消灭威胁** - 优先消灭近处的敌人
- 🔍 **全面观察** - 注意所有敌方坦克
- 📏 **保持距离** - 不要离敌人太近
- ⚡ **果断决策** - 快速做出战斗决策
